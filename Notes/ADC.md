# ADC 
 
MXT2088, an ad9288 clone, 100MSPS, dual channel, parallel output 
 
Pin connections: 
- D0A-D7A -> MCU pins 81-88 (PD0-PD7) 
- D0B-D7B -> MCU pins 55-62 (PD8-PD15) 
- DFS -> HC595/2 pin 1 (Qb) 
- S1 -> HC595/2 pin 2 (Qc) 
- S2 -> HC595/2 pin 3 (Qd) 
- ENCA -> MCU pin 69 (PA10) 
- ENCB -> MCU pin 39 (PE9)
